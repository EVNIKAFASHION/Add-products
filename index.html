⚙️ One-time setup required:
1. Open Google Sheets → create a new spreadsheet → go to Extensions → Apps Script.
2. Delete any existing code and paste the contents of Code.gs (included in your download).
3. Click Deploy → New Deployment → Web App. Set Execute as: Me and Who has access: Anyone. Click Deploy and copy the URL.
4. Paste that URL below and click Save URL. You only need to do this once — it's saved in your browser.
5. In index.html, find the line // APPS_SCRIPT_URL_PLACEHOLDER and replace "" with your deployment URL (see the snippet in your download).
